# Portfolio Starter

This is a mobile-first personal portfolio starter for showcasing:

- CAD drafting and Revit work
- Architectural rendering
- Photography
- Web development projects

## Files

- `index.html` contains the page structure and starter portfolio copy
- `styles.css` contains the full visual design and responsive layout
- `script.js` contains the mobile navigation toggle

## Next customisation steps

1. Replace placeholder contact links in `index.html`
2. Swap visual placeholder blocks with real project images
3. Add project-specific case study text and external links
4. Deploy as a static site or convert into a Django project later if needed
